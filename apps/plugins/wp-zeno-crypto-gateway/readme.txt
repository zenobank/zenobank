Accept cryptocurrency payments securely in WooCommerce

== Description ==

**Zeno Crypto Payment Gateway** allows WooCommerce store owners to accept cryptocurrency payments seamlessly.  

== Installation ==

== Installation ==

1. Install the plugin directly through the WordPress plugin installer.  
2. Activate the plugin through the 'Plugins' screen in WordPress.  
3. Go to **WooCommerce > Settings > Payments** and enable "Zeno Crypto Payment Gateway".  
4. Configure your EVM wallet to start receiving payments.
